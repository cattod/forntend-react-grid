# catodgrid
A react typescript module that returns the plural form of any noun
## Installation 
```sh
npm install catodgrid --save

```
## Usage
### React TypeScript
```typescript
import React , {Component} from "react"
import { CatodGrid } from 'catodgrid';

export class Example extends Component {
    render(){
        return(
            <CatodGrid/>  
        )
    }
}
```
